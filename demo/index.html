EUR-Lex API Demo

Standalone demo of the Cellar REST API and SPARQL endpoint — no build step, no dependencies. Not integrated with the main app.

1. Fetch Law by CELEX ID (Cellar REST API)

Endpoint: GET https://publications.europa.eu/resource/celex/{id}
No authentication required. 



  

    Known CELEX IDs to try
    
32016R0679  — GDPR (Regulation 2016/679)
32022R2065  — DSA (Regulation 2022/2065)
32022R1925  — DMA (Regulation 2022/1925)
32024R1689  — AI Act (Regulation 2024/1689)
32023R2854  — Data Act (Regulation 2023/2854)
32022R0868  — Data Governance Act (Regulation 2022/868)

  







  
2. Search Laws by Keyword (SPARQL Endpoint)

  

    Endpoint: POST http://publications.europa.eu/webapi/rdf/sparql

    Returns CELEX IDs + titles. No authentication required, but may be CORS-restricted in browsers.
